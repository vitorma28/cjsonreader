# Arquitetura utilizada

## Visão geral

JSONReader utilizará Map para armazenar os valores do JSON.

## Módulos utilizados

- Map -> Serve para representar a estrutura de objetos do javascript;
- JSONReader -> Reader/Writer entre JSON e Map.

## Diagramas de código

Diagramas estarão armazenadas na pasta `diagrams` em `map` e `jsonreader` contendo diagramas de códigos de seus respectivos diretórios.
