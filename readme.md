# JSON Reader em C

Programa que transforma uma string JSON em uma struct map.

# Diretórios principais

Os nomes de arquivos deverão estar em Snake Case.

## tests

Pasta para testes de cada componente.

Deverão iniciar com o nome do componente.

## src

Pasta que contém todos os componentes separados em subdiretórios.

Os diretórios de componentes deverão estar totalmente em minúsculo. Ex.: `JSONReader` -> `jsonreader`.
