# Requisitos de software

## Nome

C JSON Reader

## Descrição curta

Um leitor de strings `JSON` que o transforma em um `Map`.

## Requisitos Funcionais

1. Transformar string `JSON` em `Map`;
2. Transformar arquivo `JSON` em `Map`;
3. `Map` deve ter funções básicas: `set`, `get`;
4. Todos os tipos do `JSON` deverão ser suportados: `String`, `Number`, `Boolean`, `Null`, `Object` e `Array`.
5. Função para fazer o reverso: transformar um `Map` em `JSON`.

## Requisitos Não Funcionais

1. `Map` deve ser do tipo `LinearMap`;
2. As chaves de `Map` são `strings`.
